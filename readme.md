## Requirements
[.Net Core SDK 3.1.302](https://dotnet.microsoft.com/download/dotnet-core/3.1)

Test dotnet version :

```bash
  dotnet --version
  $ 3.1.302
```

Restore solution dependencies

```bash
   dotnet restore
```

compile solution projects

```bash
   dotnet build
```

To run unit tests:

```bash
   dotnet test
```


change to application directory AttempCheckerApplication and run:

```bash
   dotnet run
```
